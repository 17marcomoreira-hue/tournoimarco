Installer l’application Tournoi

Une fois installée, l’app s’ouvre en plein écran et vos tournois restent enregistrés sur cet appareil.
⬅ Retour à l’accueil

Mac (Chrome / Edge)

  1. Ouvrez l’app dans Chrome (ou Edge).
  2. Cliquez l’icône Installer dans la barre d’adresse ⬇️ (ou menu ⋯ → Installer l’application).
  3. Lancez ensuite l’app depuis le Dock ou le Launchpad.
Conseil : gardez le même navigateur et évitez la navigation privée pour conserver les tournois.

PC (Windows – Chrome / Edge)

  1. Ouvrez l’app dans Chrome (ou Edge).
  2. Cliquez Installer (icône +) dans la barre d’adresse ou menu ⋯ → Installer.
  3. Ouvrez l’app depuis le menu Démarrer ou le raccourci créé.
Important : ne supprimez pas les “données du site” si vous souhaitez garder les tournois.

Android (Chrome)

  1. Ouvrez l’app dans Chrome.
  2. Appuyez sur le menu ⋮ (en haut à droite).
  3. Sélectionnez Installer l’application ou Ajouter à l’écran d’accueil.
  4. Lancez l’app depuis l’icône sur l’écran d’accueil.
Astuce : si “Installer” n’apparaît pas, rechargez la page et réessayez.

iPhone / iPad (Safari)

  1. Ouvrez l’app dans Safari (pas Chrome).
  2. Appuyez sur Partager (carré avec flèche).
  3. Choisissez Sur l’écran d’accueil.
  4. Validez avec Ajouter, puis lancez l’app depuis l’icône.
Note : si l’option n’apparaît pas, ouvrez le lien directement dans Safari (pas depuis une autre app).

Conservation des tournois : vos tournois sont enregistrés localement (sur l’appareil + navigateur). Ils restent disponibles d’une leçon à l’autre tant que vous ne supprimez pas les données du site et que vous n’utilisez pas la navigation privée.
